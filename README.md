# Emoji Searcher

Search emoji via [Emojipedia](https://emojipedia.org/).

### Usage

![search](./screenshot/search.jpg)

- `emoji <keyword>` to search
- `CMD + C` to copy emoji
- `enter` to open _Emojipedia_ / `shift` to preview _Emojipedia_
